# PM_2018_Miyako_Fes
 Code of projection mapping at Miyako Fes 2018.

# Description

## --- wall ---

### PM_master
 コントローラ（親機）のプログラム

### PM_slave
 スレーブ（子機）のプログラム (A,B1,B2,C面)

## --- floor ---

### PM_floor_controller
 床面 コントローラのArduinoプログラム

### PM_floor
 床面 OpenFrameworksのプログラム

### PM_floor_testcord
 コントローラとOpenFrameworks間の通信を確認するためのテストプログラム
